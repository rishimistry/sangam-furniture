# Sangam Furniture

A modern e-commerce website for Sangam Furniture, showcasing premium handcrafted Indian furniture since 1998.

## About

Sangam Furniture is a furniture retail platform that blends timeless Indian craftsmanship with contemporary design. The website features:

- Product catalogue with categories (Living Room, Bedroom, Office, Dining, Custom, Storage)
- Individual product pages with detailed specifications
- Customer testimonials and reviews
- Contact and showroom information
- WhatsApp integration for customer support
- Responsive design optimized for all devices

## Tech Stack

- **Framework:** Next.js 16 with App Router
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Vercel

## Features

- Server-side rendering for SEO optimization
- Dynamic product pages with image galleries
- Category-based product filtering
- Mobile-first responsive design
- Animated UI components
- SEO-friendly with sitemap and robots.txt
- PWA-ready with manifest.json
